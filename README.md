# fml

Fml is a toy regular expression engine written in C.

## Build

### Production build

```
make fml
```

### Debugging build

```
make fml_debug
```

## Tests

### Parser tests

```
make run-parser-tests
```

### Instructions tests

```
make run-instructions-tests
```

### Compiler tests

```
make run-compiler-tests
```

## Development

### Format

```
make format
```
